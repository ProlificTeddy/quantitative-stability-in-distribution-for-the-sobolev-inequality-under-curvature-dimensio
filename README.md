# Quantitative Stability in Distribution for the Sobolev Inequality under Curvature Dimension Condition

[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](https://opensource.org/licenses/MIT)
[![Python Version](https://img.shields.io/badge/Python-3.8%2B-brightgreen.svg)](https://www.python.org/downloads/)
[![arXiv](https://img.shields.io/badge/arXiv-2607.05304v1-orange.svg)](https://arxiv.org/pdf/2607.05304v1)

This repository contains a Python implementation of the key results and methods presented in the paper **"Quantitative stability in distribution for the Sobolev inequality under curvature dimension condition"** by Max Fathi and Ivan Yuri Violo. The implementation focuses on the quantitative stability properties of the critical Sobolev inequality in ${\sf CD}(N-1,N)$ metric measure spaces, as well as related results for the log Sobolev inequality and spectral gap.

---

## 📜 Overview of the Paper

### Core Idea
The paper investigates **quantitative stability properties** of the Sobolev inequality in spaces satisfying the curvature-dimension condition ${\sf CD}(N-1, N)$. Specifically:
- If the optimal constant for the Sobolev inequality is close to that of the round sphere, the cumulative distribution of any almost extremal function is shown to be close (in Wasserstein distance) to that of an **Aubin-Talenti bubble** on the round sphere.
- Similar results are derived for the **log Sobolev inequality** and the **spectral gap**, under various curvature and dimension assumptions.
- The stability results are **quantitative** with sharp exponents, providing precise bounds on the deviation.

### Key Contributions
1. **Quantitative Stability**: The paper provides rigorous bounds that quantify how close almost extremal functions are to the optimal configuration.
2. **Wasserstein Distance**: The use of Wasserstein distance as a measure of closeness between distributions is central to the results.
3. **General Framework**: The results are established under the ${\sf CD}(N-1, N)$ curvature-dimension condition, making them applicable to a wide class of metric measure spaces.

---

## ⚙️ How It Works

This repository implements the following components from the paper:

1. **Sobolev Inequality Stability**:
   - Computes the deviation between the cumulative distribution of a given function and the Aubin-Talenti bubble on the round sphere.
   - Uses Wasserstein distance to quantify this deviation.

2. **Log Sobolev Inequality**:
   - Implements stability results for the log Sobolev inequality under curvature and dimension constraints.

3. **Spectral Gap Analysis**:
   - Analyzes the spectral gap in ${\sf CD}(N-1, N)$ spaces and computes quantitative stability results.

### Mathematical Background
- **Sobolev Inequality**: A fundamental inequality in functional analysis that relates norms of a function and its gradient.
- **Aubin-Talenti Bubble**: A family of functions that achieve equality in the Sobolev inequality on the round sphere.
- **Wasserstein Distance**: A metric on probability distributions, used here to measure the "distance" between the distributions of functions.

The implementation is designed to:
- Numerically compute Wasserstein distances.
- Verify quantitative stability properties for user-defined functions in ${\sf CD}(N-1, N)$ spaces.

## 🚀 Usage Instructions

### Prerequisites
- Python 3.8 or higher
- Required Python libraries:
  - `numpy`
  - `scipy`
  - `matplotlib`

Install the dependencies using:
```bash
pip install -r requirements.txt
```

### Running the Script
The main implementation is contained in `implementation.py`. To run the script, use the following command:
```bash
python implementation.py --function <function_name> --space <space_type> --params <parameters>
```

#### Arguments
- `--function`: The function to analyze. Options include:
  - `aubin_talenti`: The Aubin-Talenti bubble.
  - `custom`: A user-defined function (requires additional input).
- `--space`: The type of metric measure space. Options include:
  - `sphere`: The round sphere.
  - `euclidean`: Euclidean space.
- `--params`: Additional parameters for the function or space (e.g., dimension `N`).

#### Example Usage
1. **Analyze an Aubin-Talenti Bubble on the Sphere**:
   ```bash
   python implementation.py --function aubin_talenti --space sphere --params N=3
   ```

2. **Analyze a Custom Function in Euclidean Space**:
   ```bash
   python implementation.py --function custom --space euclidean --params "N=3,alpha=2"
   ```

### Output
The script outputs:
- The Wasserstein distance between the cumulative distribution of the input function and the Aubin-Talenti bubble.
- Plots comparing the distributions.
- Numerical stability bounds as derived in the paper.

## 🛠️ Implementation Details

The implementation follows these steps:
1. **Function Sampling**:
   - Sample the input function on the specified metric measure space.
2. **Cumulative Distribution**:
   - Compute the cumulative distribution of the function.
3. **Wasserstein Distance**:
   - Calculate the Wasserstein distance between the input function's distribution and the Aubin-Talenti bubble.
4. **Stability Bounds**:
   - Compute and display the quantitative stability bounds derived in the paper.
